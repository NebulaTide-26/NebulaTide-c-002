##### 特别说明：
1. 使用“DEV C++”打开c语言程序时，请将  
"工具-编译器选项-显示-<ID 27071 translation missing>"（以6.3版本为例）  
取消勾选，以正常显示中文注释
2. 使用“DEV C++”编译c语言程序时，若出行中文输出乱码，请将
"工具-编译选项-编译器-编译时加入以下命令"
勾选，并填写
"-fexec-charset=GBK"（去掉引号）
